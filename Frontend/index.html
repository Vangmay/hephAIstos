HephAIStos

An AI Coding Agent built for your CLI

Overview

HephAIStos is a modular, extensible agent that combines tool implementations, a planner, and an agent core to help you code faster and safer.

Key Features

  • Tool Registry for easy extension
  • Rule-based (and soon LLM-based) planning
  • Sandboxed file operations for safety
  • Simple CLI entry point

Get Started

  1. Clone the repo
  2. pip install -r requirements.txt
  3. python HephAIstos.py
