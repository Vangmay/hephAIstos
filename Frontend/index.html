About HephAIStos

HephAIStos is designed around three main concepts: Tool Implementations, Planner, and Agent Core. The architecture aims to be modular, testable, and easy to extend with new tools or planning strategies.

Architecture Overview

  • Modular Design: Easy to extend with new tools
  • Testable: Each component can be tested independently
  • Extensible: Support for new planning strategies
  • Safe: File sandboxing and safety helpers

Get Started

  1. Clone the repository from GitHub
  2. Install dependencies: pip install -r requirements.txt
  3. Create your .env file with your API keys
  4. Run: python hephaistos.py
